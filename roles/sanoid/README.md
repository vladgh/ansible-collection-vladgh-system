## Role Variables

Available variables are listed below, along with default values (see `defaults/main.yml`)

```yaml
# Templates can be used to define common behavior
sanoid_templates:
  default:
    frequently: 4
    hourly: 24
    daily: 7
    monthly: 12
    yearly: 1
    autosnap: true
    autoprune: true
# The datasets you want to configure
sanoid_datasets: {}
```
